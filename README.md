1. 概述
ShortVideo 是七牛推出的一款适用于 Android ，IOS 平台的短视频 SDK，提供了包括美颜、滤镜、水印、断点录制、回删、本地存储，云端存储在内的多种功能，还可以支持高度定制以及二次开发。


2. 功能列表
视频采集
音频采集
自动 / 手动对焦
视频 H.264 硬编码
音频 AAC 硬编码
实时美颜
实时滤镜
支持第三方美颜
支持第三方滤镜
自定义时长
自定义码率
自定义分辨率
支持 1:1 录制
断点拍摄
回删视频
视频水印
视频存为 .mp4 格式
上传至七牛云存储
支持 armv7, arm64, i386, x86_64 体系架构
